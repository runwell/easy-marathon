# easy marathon

Easy-Marathon is a collection of tools designed to help marathon runners plan, manage, and track their training.

https://runwell.github.io/easy-marathon

## Tool 1. Race Calendar

Generates a personalized marathon race calendar based on your preferences, including race name, date, start time, and time zone.

## Tool 2. Training Log

Provides a comprehensive overview of training activities, including running, cycling, and strength workouts.

## Tool 3. 50 Marathon Tracker

Helps runners track their progress toward completing marathons in all 50 U.S. states.

* ``50marathon-tracker/collect_marathon_data.py`` script generated by Gemini 3 to collect marathon data from https://findmymarathon.com.
* ``50marathon-tracker/collected-us-marathon.csv`` contains the collected marathon data.

## Tool 4. Pace Calculator

Calculates target paces for various race distances based on your goal marathon time.
